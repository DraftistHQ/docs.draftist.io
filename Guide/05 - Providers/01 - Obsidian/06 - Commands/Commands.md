All Draftist commands are available through Obsidian's command palette (summoned via `Ctrl/Cmd + P`). Some are also accessible via the file explorer context menu. ^6a0daf

## Content ^ca7043

These commands work with any content type — blog posts, doc pages, and any future modules. ^e937c2

- **Insert image** — picks an image from your vault or filesystem, copies it into the content's `images/` folder, and inserts it at the cursor. ^4270d9
- **Insert gallery** — same as above, but for multiple images arranged in a gallery layout. ^7279ab
- **Normalize images** — moves images into the `images/` folder and normalizes filenames. ^70f98b
- **Preview and publish** — uploads assets, publishes the current note as a draft, and opens a preview in the browser. ^32ee3c
- **Normalize frontmatter** — sorts frontmatter fields into a consistent order and adds any missing fields. ^dfb9ab

## Blog Module ^ae2d61

- **Create new blog post idea** — creates a new post with `Idea` status. You'll pick a site, enter a title, and optionally a description. ^ae0caf
- **Create new blog post draft** — same as above, but with `Draft` status. ^d32140
- **Set cover image** — sets a cover image for the current post, with optional credit text and link. ^af2848

> [!NOTE] File explorer context menu items
> - Add post idea
> - Add post draft 
^c7a99c

## Docs Module ^70c7fa

- **Create new doc page** — creates a new page. You'll pick a location in the tree, enter a title, and optionally a description. ^e48879

> [!NOTE] File explorer context menu items
> - Add page before
> - Add page after
> - Add child page
> - Move up
> - Move down
^db302e

## Maintenance ^72b009

- **Copy debug info** — copies your OS, Obsidian, and plugin versions to the clipboard for support requests. ^d9357a
