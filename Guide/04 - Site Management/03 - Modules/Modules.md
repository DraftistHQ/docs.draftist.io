The modules page lists all modules on your site. Each card shows the module type (Blog, Docs, etc.), its name, and its live URL. ^287f2c

> [!warning]
> For now, only the Blog module is available, and it's created automatically with your first site. Multiple module support is on the way.
^4ee5eb

## Editing a module ^fc02ac

Click the settings icon on a module card to open the edit modal. You can change two things: ^ea99fc

- **Name** — displayed in your site's navigation. Must be unique across the site. ^0de7b7
- **Slug** — the URL path for this module (e.g., `my-site.com/blog`). Must start and end with a letter or number, and can contain lowercase letters, numbers, hyphens, and underscores. ^e93433

If you change a module's slug, old URLs automatically redirect to the new address — so existing links and bookmarks won't break. ^bc7982

> [!info]
> More configuration options — such as default metadata and OG tag settings — will be added in future updates.
^f0eb32
