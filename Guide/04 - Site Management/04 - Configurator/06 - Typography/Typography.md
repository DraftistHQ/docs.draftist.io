The Typography section controls the fonts used across your site. There are three font faces: ^52f6f1

- **Base** — body text ^dc31af
- **Head** — headings ^ec509a
- **Mono** — code and monospace content ^2f11f9

For each face, you can pick from built-in system fonts or upload a custom font file. Custom fonts support a fallback font stack in case they fail to load. ^25fa7f

> [!warning]
> Currently, only a single font file per face is supported. Multiple variants (bold, italic, etc.) will be available in a future update.
^29e5c2

You can also adjust the **base font size** — the global size that all other text sizes scale from. ^934d2f
