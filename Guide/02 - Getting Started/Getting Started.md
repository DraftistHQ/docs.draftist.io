Here's the shortest path to getting your site up and running. ^552088

## 1. Sign up ^21fdbc

Draft42 is currently in private beta. Join the waitlist from the [main page](https://draft42.io/) — we'll send you an invite when a spot opens up. ^f46882

During registration, you'll pick a username — it serves as your platform handle and becomes part of your site's address (e.g., `username.draft42.io`). ^148313

## 2. Create a site ^45f78e

Once you're in, we will ask you to create your first site. You'll need to provide a name — it's used as the site title, default logo, and in metadata. Pick something that fits your site's purpose: your own name for a personal site, your company name for a company blog, etc. ^b8fa16

A site and a [Blog module](Guide/04%20-%20Site%20Management/03%20-%20Modules/Modules.md) are set up automatically. ^3858b5

> [!info]
> For now, every site is created with a Blog module by default. Once we ship more modules, you'll be able to select the modules you need.
^2a0e25

## 3. Install the Obsidian plugin ^d97772

Install the Draft42 plugin in Obsidian. See [Installation](Installation.md) for instructions. ^0a053d

## 4. Connect to your account ^ec8978

Open Obsidian and sign in to Draft42 through the plugin. See [Setup](Setup.md) for details. ^e573d1

## 5. Write your first post ^5ebb6b

Create a new blog post in Obsidian using the [`Draft42: Create new blog post draft` command](Commands.md#^ae2d61). Give it a title, write some content, and add an optional description ([we suggest not skipping it!](Blog.md#^9d1ac8)). ^7d1515

> [!tip]
> Open the command palette with `Ctrl/Cmd + P` and start typing the command name to find it.
^4ff2b0

## 6. Preview and publish ^5c899b

Run [`Draft42: Preview and publish` command](Commands.md#^32ee3c) to push your draft to the site and preview it in the browser. When it looks good, publish it to make it live. ^4bed3a

---

Now that you've published your first post draft, head to [Concepts](Concepts.md) to learn more about how Draft42 works — sites, modules, domains, drafts, and more. ^0f37dc
